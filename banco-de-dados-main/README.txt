# 📚 Sobre o projeto

O **Sistema Escolar** é uma aplicação web desenvolvida com o objetivo de facilitar o gerenciamento das informações de uma instituição de ensino.

O sistema foi criado para oferecer uma interface simples, intuitiva e organizada para diferentes perfis de usuários, permitindo o acesso às funcionalidades específicas de:

- 👨‍🎓 Alunos
- 👨‍🏫 Professores
- 👨‍💼 Administradores

Cada perfil possui sua própria área de acesso, proporcionando uma melhor organização do ambiente escolar.

---

# 🎯 Objetivo

O projeto tem como finalidade centralizar o gerenciamento escolar em um único sistema, permitindo que usuários possam acessar suas informações de forma rápida e segura.

---

# ✨ Funcionalidades

## 🔐 Login

- Autenticação de usuários
- Redirecionamento conforme perfil
- Interface moderna

---

## 📝 Cadastro

Permite realizar o cadastro de novos usuários.

Campos personalizados para cada perfil.

---

## 👨‍💼 Área do Administrador

O administrador possui acesso completo ao sistema.

Possíveis funcionalidades:

- Cadastro de usuários
- Gerenciamento geral
- Controle das informações

---

## 👨‍🏫 Área do Professor

O professor possui acesso ao seu painel para gerenciamento acadêmico.

Possibilidades futuras:

- Lançamento de notas
- Frequência
- Turmas
- Avisos

---

## 👨‍🎓 Área do Aluno

Área destinada ao aluno para consulta das suas informações.

Exemplos:

- Dados pessoais
- Notas
- Frequência
- Informações escolares
